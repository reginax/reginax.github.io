--- title: NOTE2NOTE permalink: "note2note/index.html" layout: default ---

Open-source note sharing Android app with Parse backend. Accessing and querying images from database with ParseQuery and ImageView. Improvements: OCR with possible CamScanner API integration for cleaner interface; point-tracking system to encourage note sharing, where each note file contributed is a chance to view a note from the database; enter class info per university into database; add teacher privileges to prevent cheating or answer sharing.
